# Flutter를 선택하게된 이유

- React-Native의 개발 속도
- 이해를 알기 어려운 재랜더링과 퍼포먼스 이슈
- React과 유사한 ionic의 경험

## Flutter의 진입 장벽

- Dart라는 특이한 언어
- 적응하기 어려운 Widget이라는 UI 표현 방식
- Web 이라는 함정

- 수많은 Flutter Widget, 다양한 gallery, 그러나 간편한 디자인 시스템 적용
- Getx라는 우수한 라이브러리와 Firebase 공식 지원
